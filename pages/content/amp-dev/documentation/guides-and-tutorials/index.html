Get started with AMP!

Discover your journey

The AMP Project started as a web component library designed to make mobile websites load instantaneously. And while we've stuck to our original ideals of speed and great user experiences, AMP has epxanded its capabilites beyond the mobile web.

{% do doc.styles.addCssFile('/css/components/atoms/format-tags.css') %}

Today, AMP can build responsive and interactive websites, immersive full-screen stories, secure and engaging ads and dynamic email.
